// GET STARTED //////////////////////////////////////////////

- In your code, add "using GoodEnough.TextToSpeech;" to each class where you want to use Text To Speech.
- Use "TTS.Speak("Your text");" to make your device speak any text.
- This plugin works only on iOS or tvOS devices. You will need to make a build and start it on an iPhone, iPad or Apple TV device to hear the speech.
- Check out the links below for full documentation.
